# Market

A modern iOS mock shopping app built with **SwiftUI** and **SwiftData**, focused on clean **MVVM** architecture.

## Goals
- Build a production-style iOS app using SwiftUI and SwiftData  
- Practice clean MVVM architecture  
- Implement real-world API integration  
- Design a scalable project structure  
- Improve UI polish and app performance  
- Have fun  

## Features
- Product browsing  
- Categories  
- Search  
- Cart  
- Favorites  
- SwiftData persistence  
- API integration (Platzi Fake Store API)

## Tech Stack
- Swift  
- SwiftUI  
- SwiftData  
- MVVM  
- async/await networking
